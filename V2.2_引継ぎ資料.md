## 2. V2.1 で完成した内容

### フィルターパネル（上部・全タブ共通）

```
🎯 解析対象フィルター  [釣り場: すべて] [釣法: すべて]  ▼
────────────────────────────────────────────────────────────
  🗺 釣り場（6グループ）  |  🎣 釣法（12グループ）
  ▼室戸沖 [全解除]       |  ▼泳がせ釣り系 [全解除]
    □室戸沖 □室戸沖周辺…  |    □泳がせ釣り □泳がせ釣り(手巻き)…
  ▼室戸沖近場 [全解除]   |  ▼ライト泳がせ釣り系 [全解除]
  …                       |  …
────────────────────────────────────────────────────────────
  [✅ この条件で解析]  [↺ リセット（全選択）]
```

**実装関数：**  
- `initFilterPanel(data)` → `buildGroupedCBList('cb-spots', ..., SPOT_GROUPS)` / `buildGroupedCBList('cb-methods', ..., METHOD_GROUPS)`
- `buildGroupedCBList(containerId, allItems, groupDefs)` → グループチップUI生成
- `applyFilter()` → 釣り場・釣法絞り込み、GFオブジェクト更新
- `resetFilter()` → 全選択に戻す
- `updateBadge()` → バッジ表示更新

**グループ定数：** `SPOT_GROUPS`（6グループ）、`METHOD_GROUPS`（12グループ）

### 魚種・海域選択フィルター（魚種解析タブ内）

```
🐟 解析魚種・潮流データ海域選択フィルター  [魚種: 未選択] [海域: 室戸沖]  ▼
────────────────────────────────────────────────────────────────────────────
  🐟 解析する魚種（16グループ）  |  🌊 潮流データの海域
  ▼ブリ系 [全選択]               |  全選択 全解除
    □ブリ □ブリ（デカブリ）…    |  □室戸沖 □北西 □西 □東 □北東
  ▼カンパチ系 [全選択]           |
  …                               |
────────────────────────────────────────────────────────────────────────────
  [✅ この条件で解析]  [↺ リセット（全解除）]
```

**実装関数：**  
- `toggleSPFilter()` → sp-filter-body 開閉
- `applySPFilter()` → 魚種・海域をhidden selectに同期、`loadSpeciesProfile()` 呼び出し
- `resetSPFilter()` → 全解除、海域を室戸沖に戻す
- `updateSPBadges(spChecked, areaChecked)` → バッジ更新
- `refreshSpeciesCBPanel(profiles)` → analysis_result.jsonのprofilesからチェックボックスを再構築
- `selectAllSPCB()` / `selectNoneSPCB()` / `selectAllAreaCB()` / `selectNoneAreaCB()`

**グループ定数：** `SPECIES_GROUPS`（16グループ）

**重要：** 魚種チェックボックスは即時解析せず、「この条件で解析」ボタン押下時のみ解析実行。

## 3. 確定デザイン仕様（変更禁止）

### フィルターパネル共通デザイン
- バー：`background:var(--surface); border-bottom:1px solid var(--border)`（フルワイド、開閉式）
- グループヘッダー：`▼矢印 + グループ名 + 全解除/全選択ボタン（小型）`
- チップ：`.cb-chip` クラス（ラウンドボーダー、チェック付き、選択時アクセントカラー）
- グループ折りたたみ：ヘッダークリックで `.cb-group-chips.collapsed` トグル
- コンテナ：`.cb-list-grouped`（縦並び flex-direction:column、max-height:260px スクロール）

### CSSクラス構成
```
.cb-list-grouped   → グループコンテナ
.cb-group          → グループ1個分
.cb-group-header   → ▼ + ラベル + ボタン行
.cb-group-arrow    → ▼▶ アイコン
.cb-group-label    → グループ名テキスト
.cb-group-btn      → 全解除/全選択ボタン
.cb-group-chips    → チップ一覧（flex-wrap:wrap）
.cb-group-chips.collapsed → display:none
.cb-chip           → 個別チップ（label要素）
```

## 7. 開発上の注意事項

1. **ファイルが大きいため直接読み込み不可** → `grep -n` や `sed -n 'N,Mp'` でセクションを切り出して確認する
2. **変更はPythonの文字列置換で行う** → 置換前に `assert old_str in html` で存在確認を必ず行う
3. **作業は `/tmp/` にコピーしてから行い、最後に `mnt/` にコピーして保存**
4. **保存後は必ず `present_files` でリンクを提示する**
5. **解析対象フィルター（釣り場・釣法）のデザインは確定済み → 変更禁止**
