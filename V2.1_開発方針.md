# 釣果データ解析ソフト V2.1 開発方針

作成日：2026-04-15  
対象バージョン：V2.1（dashboard.html）

---

## 1. 開発の目的

室戸沖の釣果データを多角的に解析し、「いつ・どんな条件で・どの魚が釣れるか」を視覚的に把握できるダッシュボードを構築する。データの蓄積とともに解析精度が向上する仕組みを前提とし、現場（船上・岸壁）でのスマートフォン利用も考慮したUI設計を行う。

---

## 3. 確定デザイン仕様（V2.1基準）

### フィルターパネル共通仕様

- **バー**：ダークブルー背景（`var(--surface)`）、フルワイド、クリックで開閉（▼▲）
- **バッジ**：選択状態をトグルバーに常時表示。全選択時は通常色、一部選択時はオレンジ色でアクティブ表示
- **グループヘッダー**：▼矢印 ＋ グループ名 ＋「全解除/全選択」ボタン（小型）
- **チップ**：ダークブルー背景（`var(--surface2)`）、ラウンドボーダー、チェックボックス付き。選択時はアクセントカラーのボーダー
- **折りたたみ**：グループヘッダークリックで配下チップを個別に折りたたみ可
- **適用**：「✅ この条件で解析」ボタンを押した時点で解析が実行される
- **リセット**：フィルターは「全選択に戻す」、魚種は「全解除」

## 4. データ構造

解析は `analysis_result.json` から読み込む。  
`analyze.bat` → `analyze_engine.py` で生成。

主要フィールド：
- `all_spots`：釣り場一覧
- `all_methods`：釣法一覧
- `species_profiles`：魚種別プロファイル（釣果日数・水温・流速等）
- `records`：釣果レコード（spot / method / species / 日付 等）
- `db_info`：DB件数情報
