# 釣果データ解析ソフト V2.1 取り扱い説明書

作成日：2026-04-15

---

## 起動方法

1. `analyze.bat` をダブルクリック → `analysis_result.json` が生成される
2. `dashboard.html` をブラウザで開く
3. 自動的に `analysis_result.json` を読み込んで解析開始

> JSONが見つからない場合は「データ読込」タブからファイルを手動選択する

---

## 画面構成

```
┌─ ヘッダー（タイトル・件数表示）────────────────────────┐
├─ 解析対象フィルター（釣り場・釣法の絞り込み）──────────┤
├─ タブナビ：魚種解析 / 釣れ予測 / 水温TL / 流速TL /   │
│           月齢TL / 5地点海流 / 魚種比較 / データ読込  │
├─ 各タブのコンテンツ────────────────────────────────────┤
```

---

## 解析対象フィルターの使い方

画面上部の「🎯 解析対象フィルター」バーをクリックして開く。

### 釣り場の選択

- 6グループ（室戸沖 / 室戸沖近場 / 室戸港・磯 / 高知西部 / 愛媛 / 遠征（小笠原））
- グループ名の左「▼」をクリック → グループを折りたたむ
- グループ右の「全解除/全選択」→ グループ内を一括切替
- 列上部の「全選択/全解除」→ 全釣り場を一括切替

### 釣法の選択

- 12グループ（泳がせ釣り系 / ジギング系 / SLJ系 / タイラバ系 など）
- 操作方法は釣り場と同じ

### 解析の実行

「✅ この条件で解析」ボタンを押す → 絞り込み結果で全タブが更新される  
「↺ リセット（全選択）」→ すべての選択を戻す

バー上のバッジ（「釣り場: N/32」など）で現在の選択状態を確認できる。

## 魚種解析タブの使い方

### 魚種・海域の選択

「🐟 解析魚種・潮流データ海域選択フィルター」バーをクリックして開く。

**解析する魚種（左列）**

- 16グループで整理（ブリ系 / カンパチ系 / マグロ系 / 鯛系 など）
- 複数の魚種を同時に選択可能（比較表示される）
- グループ▼で個別折りたたみ可

**潮流データの海域（右列）**

- 室戸沖 / 北西 / 西 / 東 / 北東 から選択
- 複数選択可（Ctrl不要、チップをクリックするだけ）
- どの海域の水温・流速データと照合するかを指定する

「✅ この条件で解析」を押すと解析が実行され、パネルが閉じる。  
バッジに「魚種: 2/57」「海域: 室戸沖」のように表示される。

### 表示内容（1魚種選択時）

- サマリーカード（釣果日数・記録件数・最多釣果月・水温・流速・波高・月齢）
- 水温帯別釣果分布
- 月別釣果数
- 流速帯別釣果分布
- 潮汐別釣果分布
- 水温トレンド（釣れた日の水温変化方向）
- 流れの方向別分布（棒グラフ＋コンパスチャート）
- 5地点海流プロファイル
- 海況条件プロファイル（全期間平均との比較）
- 釣り方 / 天気 / 月相 の統計

### 表示内容（複数魚種選択時）

- 魚種比較テーブル（釣果日数・釣果記録）
- 各チャートに複数魚種が重ねて表示される

## 釣れ予測タブ

現在の海況データ（水温・流速など）を入力し、各魚種の「釣れやすさ」スコアを計算する。

---

## 水温・流速・月齢タイムラインタブ

期間中の水温・流速・月齢の推移と釣果記録をオーバーレイ表示する。

---

## 5地点海流タブ

室戸沖5地点（室戸沖・北西・西・東・北東）の海流データを比較表示する。

---

## 魚種比較タブ

2魚種を選んで、各種指標を並べて比較する。

---

## データ更新の手順

1. 釣果データ収集ソフト（V6系）で新しいデータを追加
2. `analyze.bat` をダブルクリックして `analysis_result.json` を再生成
3. `dashboard.html` をブラウザで再読み込み（またはデータ読込タブから手動選択）

---

## 注意事項

- `dashboard.html` と `analysis_result.json` は同じフォルダに置くこと
- ブラウザはChrome / Edge 推奨
- スマートフォンでも利用可能（フィルターパネルは固定幅で表示）
